# Noju.io

Noju.io is a modern, visually engaging company website designed to present services in an abstract and interactive way. Leveraging a dynamic particle system, the platform creates a unique user experience while providing practical business functionalities. Users can seamlessly schedule calls with the marketing team, enabling effective customer relationship management and service inquiries.

### Key Highlights

* Abstract Service Presentation: Services are displayed in a visually creative and abstract format, enhanced by an interactive particle system.

* Direct Carrier Billing: Simplifies billing through direct integration with carriers, streamlining payment processes.

* Content Licensing: Offers tailored content licensing solutions to meet the needs of diverse businesses.

* Digital Marketing Strategy: Supports strategic digital marketing planning and implementation to drive business growth.

* Schedule a Call: Easy-to-use scheduling functionality allows users to book calls with the marketing team, enhancing client engagement and relationship management.

### Platform Features

* Abstract Service Listings: Interactive, visually appealing service presentation using dynamic particles.

* Direct Carrier Billing: Integrated billing solutions directly through carriers for convenience and efficiency.

* Content Licensing: Manage content rights and licenses easily with flexible options.

* Digital Marketing Strategy: Develop and execute marketing strategies with measurable results.

* Schedule a Call: Integrated booking system to schedule calls with the marketing team for consultations and support.

### Technology Stack

| Technology / Feature    | Description                                                                                   |
|-------------------------|-----------------------------------------------------------------------------------------------|
| **Frontend**            | Built with **Next.js** and **React** for fast, interactive user interfaces. Uses **Tailwind CSS** for responsive and modern styling. |
| **Particle System**     | Custom-built particle animation engine for a dynamic and visually engaging user experience.  |
| **Scheduling System**   | Fully integrated call scheduling system, allowing users to book appointments directly with the marketing team. |
| **Backend**             | **Node.js** and **Express.js** handle server-side logic, API requests, and data management. |
| **Database**            | **MongoDB** stores booking, service, and content data securely and efficiently. |
| **Payments**            | Direct Carrier Billing integrated for seamless payment processing. |
| **Deployment & Hosting**| Hosted on **Vercel** / Cloud platforms for high availability, performance, and scalability. |
